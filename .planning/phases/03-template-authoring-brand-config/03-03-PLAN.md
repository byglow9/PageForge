## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → createTemplateAction/updateTemplateAction | markup (user-authored), name, metadataOverlay cross the trust boundary |
| Server Action → TenantClient | workspaceId derived from server session only; template.findById always filters by workspaceId |
| Client component → parse() | parse() runs client-side on markup; outputs liveSchema advisory only; authoritative parse is server-side on save |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
