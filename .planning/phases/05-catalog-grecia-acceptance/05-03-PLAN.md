## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Playwright test runner → app | Tests use real test user credentials; workspaceId resolved from session (never hardcoded) |
| Grécia template markup → LiquidJS engine | Template markup is user-authored input; must not execute JS (LiquidJS with ownPropertyOnly:true guards this) |
| ZIP export → downloaded file | index.html in ZIP must have Content-Security-Policy header injected by the export route handler |

| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-05-03-01 | Tampering / SSTI | Grécia template markup | accept | LiquidJS with outputEscape:'escape' + ownPropertyOnly:true (Phase 1 decision D-10); SSTI corpus tests pass (Phase 1 plan 03); no new attack surface introduced by authoring a real template |
