```typescript
const CSP_META = `<meta http-equiv="Content-Security-Policy" content="default-src 'none'; img-src 'self' data:; style-src 'self' 'unsafe-inline'; font-src 'self'; object-src 'none'; base-uri 'none';">`;
function injectCsp(html: string): string {
  if (html.includes("<head>")) return html.replace("<head>", `<head>\n  ${CSP_META}`);
  return `${CSP_META}\n${html}`;
}
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| browser GET /api/lps/{lpId}/export | unauthenticated route access attempt; lpId from URL is untrusted |
| route handler→S3 | server fetches S3 images — potential SSRF if URL filter is bypassed |
| exported ZIP→end user | ZIP contains HTML + images delivered to marketing user for use offline |
| CSP meta→browser | CSP is enforced by the browser when the exported HTML is opened locally |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-04-04-01 | Elevation of Privilege | GET /api/lps/{lpId}/export — unauthenticated access | mitigate | Route handler verifies session (better-auth); if no session → 401. WorkspaceMember check ensures LP belongs to authenticated user's workspace. No raw lpId lookup without workspaceId scoping. |
| T-04-04-02 | Elevation of Privilege | Cross-workspace LP export by ID | mitigate | After fetching LP by id, WorkspaceMember.findFirst({workspaceId: lp.workspaceId, userId: session.user.id}) confirms user's membership before proceeding. Return 403 if not a member. |
| T-04-04-03 | Tampering | SSRF via image URL in LP HTML | mitigate | Image URL filter: only URLs starting with process.env.S3_PUBLIC_BASE_URL are fetched server-side. External URLs are left as absolute references in the exported HTML (not downloaded). No redirect following. |
| T-04-04-04 | Spoofing | Script execution in exported HTML | mitigate | CSP meta injected: default-src 'none' with no script-src → scripts are blocked. The exported HTML is a static landing page intended to be served over a web server; the CSP prevents any injected script from executing. |
| T-04-04-05 | Information Disclosure | ZIP contains LP values / source markup exposure | accept | The exported ZIP is delivered to the user who requested it and is authenticated. The LP's values and markup are the user's own content. No cross-tenant content can appear in the ZIP (tenant isolation enforced before render). |
| T-04-04-06 | Denial of Service | Large LP with many images exhausting server memory during ZIP assembly | mitigate | archiver streams output directly (does not buffer the entire ZIP in memory). Images are fetched sequentially and appended to the stream individually. Memory usage is bounded by the largest single image size. |
</threat_model>
