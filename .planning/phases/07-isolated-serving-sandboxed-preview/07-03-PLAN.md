## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| RSC page → mintServeToken | Token is minted server-side with workspaceId from session; the iframe src (including ?t=) is sent to the browser as static HTML — the token is not generated client-side |
| Browser iframe → isolated origin | The iframe has an opaque origin due to sandbox="allow-scripts" without allow-same-origin; the browser enforces this boundary natively |
| iframe JS → parent document | Cross-origin iframe JS cannot access window.parent.document — this is enforced by the browser, not by application code |
